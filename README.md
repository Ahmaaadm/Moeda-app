# Modeo - Accounting System

Modeo is a modern desktop accounting application built with **JavaFX** and **SQLite**.  
It provides a simple yet powerful interface for managing employers, cash entries, monthly reports, and backups.  
The app is designed with a modern UI (Neumorphism / Material-inspired) for a smooth and attractive user experience.

- Automatic database backup with **Google Drive integration**.
- Modern UI design (Glassmorphism & Neumorphism styles).
- Multi-language support (Arabic/English ready).
